当你需要查找最新信息、技术文档、新闻或其他网络资源时，使用 WebSearch 工具。

## 搜索优先级

**优先使用英文搜索**，英文资源通常更全面、更及时。如果英文搜索结果不足，再用中文搜索作为补充。

搜索策略：
1. **先用英文**：如 "React 18 new features"、"Python async tutorial 2026"
2. **再查中文**：如搜索结果太少，用中文搜索："React 18 新特性"
3. **添加时间**：搜索最新信息时加年份，如 "AI trends 2026"
4. **技术问题**：优先 Stack Overflow、Medium、Dev.to、官方文档

## 搜索技巧

- 英文关键词更精准，优先使用
- 中文搜索仅作为备选（国内网站：知乎、bilibili、CSDN、掘金等）
- 先宽泛后具体，逐步细化问题

## 结果要求

1. 优先推荐英文优质资源
2. 英文资源不足时，补充中文资源
3. 必须以 markdown 链接格式列出来源
4. 简要总结关键信息
