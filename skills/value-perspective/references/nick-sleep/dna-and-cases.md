# Nick Sleep (Nomad) — 表达 DNA + 案例库

> Nick Sleep + Qais Zakaria 的 Nomad Investment Partnership（2001-2014），是现代"长期质量复利"的标杆。2014 年主动清盘还钱、转做慈善。

## 🎙 表达 DNA（5 个标记）

1. **目的地分析(Destination Analysis)**：不问"明年 EPS"，问"这门生意 10-20 年后的终局长什么样"，倒推今天。
2. **极低换手 + 极度集中**：组合常年 3-5 只占大头，换手率低到接近零——"少做事"是核心纪律。
3. **第一性原理 + 哲学化**：信里大量谈认知、耐心、噪声 vs 内容，不谈宏观择时。
4. **找"圣杯"商业模式**：终生在找一类稀有公司——**把规模优势返还给客户**的飞轮。
5. **内容 vs 噪声**：99% 的市场信息是噪声，只盯极少数决定终局的"内容"。

**典型句式**：
- "Scale economics shared."（规模经济共享——他的核心发现）
- "We are trying to find the destination, not predict the next stop."
- "The deferral of gratification... is the investor's edge."

## 🧠 核心框架：规模经济共享(Scale Economics Shared, SES)

**洞察**：多数公司规模变大后**收割**规模红利（涨价/提利润率）= scale economics **harvested**。极少数公司反向操作——**把规模带来的成本下降持续返还给客户**（更低价/更多价值）：

```
规模↑ → 成本↓ → 主动让利给客户 → 客户黏性/份额↑ → 规模再↑（飞轮）
```

这类公司（Costco / Amazon / GEICO）牺牲短期毛利，换**越来越深的护城河 + 客户信任**，复利极久。SES 是 Nomad 重仓的统一逻辑。

**与 Munger/Buffett 的连接**：Costco 是三人共同的"圣杯"案例；SES = Munger"规模优势"模型 + Fisher"长持卓越"的现代综合。

## 📚 案例

### Costco（Nomad 标志性持仓 / SES 原型）
- 把采购规模红利通过会员制返还给会员 → 续费率极高 → 越大越便宜越强。

### Amazon（约 2000s 早期重仓，逆共识）
- 当市场还把亚马逊当"不赚钱的零售"，Nomad 识别出**SES 飞轮 + Prime/AWS 长期目的地**，重仓长持，回报巨大。
- 体现"目的地分析"：看的是 2020 年的亚马逊，不是当期亏损。

### Berkshire / ASOS 等
- 同一 SES + 长持逻辑的延伸。

## 📕 局限 / 难点

- **心理极难**：SES 公司常年"看起来贵 + 短期毛利难看"，要在多年波动中**坐着不动**，绝大多数人做不到。
- **集中风险**：3-5 只占大头，单一终局判断错则重伤。
- **样本稀少**：真正的 SES 公司极少，多数"飞轮故事"是伪飞轮（只是补贴换增长，不可持续）。

## ⚠ 常见误用

- **"飞轮"当口号**：烧钱补贴 ≠ SES。SES 要求**单位经济为正**且主动让利，不是亏本买增长。
- **SES 套到所有打折公司**：降价≠规模经济共享；要看是否形成"越大越强"的结构性闭环。
- **学集中长持却没那个脾气**：Nomad 能持 10 年 + 回撤不动；散户照搬集中 = 在低点被洗。
- **忽略价格**：Nomad 也在意买入价（Amazon 是在被嫌弃时买的），不是任何价都上。
