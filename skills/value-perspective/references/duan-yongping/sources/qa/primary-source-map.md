## 精选原文锚点

### 本分：先做对的事，再把事做对

- 极短原话：「做对的事情，把事情做对。」
- 主题：本分不是保守，而是 right thing + right way；先判断事情该不该做，再判断怎么做。
- RAG 关键词：本分, 做对的事情, 把事情做对, right thing, 企业文化, 管理层诚信, 用户导向。
- 用法：分析 A 股/港股公司时，把文化和商业模式放在估值前面。管理层若反复做伤害用户/股东的事，低估值不能补偿。
- 误用：把“本分”理解成慢、不进取。网易重仓和企业经营都说明，本分可以很进取，但不做错事。

### 不为清单：stop doing 优先

- 极短原话：「stop doing list」
- 主题：知道什么不能做，常比知道做什么更重要。
- RAG 关键词：stop doing, 不为清单, 不懂不做, 不做空, 不借钱, margin, 杠杆, 能力圈。
- 用法：遇到杠杆 ETF、做空、热门题材、看不懂的新概念时，先跑“不为清单”而不是估值模型。
- 误用：把 stop doing 当作“什么都不做”。真正意思是砍掉会永久伤害本金和判断力的动作。

### 买股票就是买公司

- 极短原话：「买股票就是买公司。」
- 主题：股价只是报价，长期收益来自公司创造现金流和管理层分配现金流。
- RAG 关键词：买公司, 股票是公司一部分, 现金流, 长期持有, owner earnings, 内在价值, 公司视角。
- 用法：用户问“今天能不能买”时，先改写为“我是否愿意拥有这门生意的一部分很多年”。
- 误用：拿“买公司”掩盖仓位过大或 thesis 变坏。买的是好公司，不是买了以后永远不看。

### DCF 是思维方式，不是公式

- 极短原话：「DCF 是一种思维方式。」
- 主题：DCF 的核心是判断未来现金流的久和多，而不是小数点精确。
- RAG 关键词：DCF, 思维方式, 久和多, 自由现金流, 折现率, 安全边际, 估值, 模糊正确。
- 用法：分析茅台、苹果、腾讯、网易这类现金流公司时，用情景思考替代伪精确模型。
- 误用：为了证明想买而调折现率。段式 DCF 是约束想象力，不是制造目标价。

### 能力圈：不懂不做

- 极短原话：「不懂不做。」
- 主题：能力圈不是行业标签，而是能否解释商业模式、竞争优势、管理层行为和现金流来源。
- RAG 关键词：能力圈, 不懂不做, GE, 复杂金融, 中国消费, 互联网平台, 认错。
- 用法：遇到复杂金融、高杠杆、多元化集团、政策强变量题材时，先判定是否超出能力圈。
- 误用：看过几篇研报就以为懂。段的能力圈要求能长期跟踪、能忍受波动、能识别错误。

### 网易、苹果、茅台三案例

- 网易：极度悲观时看懂现金流和丁磊，逆向重仓，靠能力圈和长期持有兑现。
- 苹果：不按科技股理解，而按消费品、生态和企业文化理解。
- 茅台：好生意原型，提价权、品牌心智、高自由现金流、低再投入。
- RAG 关键词：网易, NTES, 丁磊, 苹果, AAPL, 茅台, 600519, 好生意, 消费品, 企业文化, 现金流。
- 用法：A 股/港股研习优先用段永平 lens；若是美国消费/平台公司，可与 Buffett lens 叠加。
- 误用：复制持仓。案例是训练“看懂生意 + 文化 + 价格 + 拿得住”，不是推荐清单。

## Digest 使用规则

- 段永平每日研习必须先问：这事在不在能力圈？是否触犯 stop doing？
- 有杠杆 ETF、做空、热点追涨新闻时，优先讲“不为清单”。
- 有消费白马、互联网平台、管理层文化新闻时，优先讲“本分 + 久和多”。
- 输出要像段的语气：大白话、少术语、重视常识和边界。
