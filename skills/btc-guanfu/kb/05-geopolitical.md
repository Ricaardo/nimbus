# 地缘冲击 (Geopolitical Shocks)

> guanfu 不直接采集新闻。本文只在用户提供地缘事件、或面板出现异常跨资产/宏观组合时使用，用于判断冲击类型和传导路径。

---

## 总原则

地缘事件对 BTC 的方向取决于它冲击什么：

| 冲击对象 | 常见 BTC 反应 |
|---|---|
| 法币体系信任 | 中期可能利好 BTC/黄金叙事 |
| 实体经济和能源成本 | 通过通胀、实际利率、美元形成逆风 |
| 交易/托管/监管通道 | 短期压制，加密内部风险上升 |
| 资本管制 | 局部溢价和避险买盘可能出现 |

不要只看新闻标题，要看二阶政策反应。

---

## 五类事件

### A. 能源供应中断

典型：战争导致油气供应冲击、航运受阻、OPEC+ 意外减产。

传导：

```
能源价格上升 → 通胀预期上升 → Fed 难以宽松 → 实际利率/美元压力 → BTC 承压
```

观察：

- `wti_crude_usd` 或 `oil_proxy_60d_trend_pct`
- `real_yield_10y_pct`
- `dxy_60d_trend_pct`
- SPX 和黄金方向

### B. 金融制裁 / 资产冻结

典型：央行储备冻结、SWIFT 制裁、大范围金融封锁。

传导：

```
法币储备可没收性上升 → 黄金/非主权资产叙事增强 → BTC 中期叙事可能受益
```

观察：

- 黄金是否上涨。
- BTC 是否相对 SPX 抗跌或上涨。
- DXY 是避险上涨还是美元信任受损。

### C. 台海/南海/亚洲军事紧张

传导可能分裂：

- 短期：风险资产去风险，BTC 跟跌。
- 中期：亚洲资本避险、USDT 溢价、矿机供应链风险。

观察：

- USDT/CNY、KRW/JPY 地区溢价需要外部数据，guanfu 不直接提供。
- BTC/SPX 相关性是否上升。
- 黄金和美元是否同涨。

### D. 银行危机 / 主权信用事件

典型：地区银行挤兑、稳定币发行方银行敞口、主权 CDS 异常。

传导：

```
银行体系信任下降 → 自托管/非银行资产叙事增强
```

但若是系统性流动性危机，BTC 也可能先跟跌。

观察：

- HY spread、VIX/VIXY、银行股外部信息。
- BTC 是否在 SPX 下跌时抗跌。
- ETF 是否继续流入。

**历史样本锚（2023-03 SVB / Signature / Credit Suisse）**：
- 2023-03-09 至 03-13 三家银行连续出险，USDC 因 Circle 在 SVB 有 33 亿美元敞口短暂脱锚至 $0.87。
- BTC 在 USDC 脱锚当日（3-11）瞬时下跌至 ~$19.5k，但 **Fed 当晚启动 BTFP 提供流动性后 5 个交易日内反弹至 $28k+**。
- 范式："加密内部脱锚冲击 + 政策快速兜底 → BTC 反而强化非银行资产叙事"。
- 不可外推到所有银行危机：当时 Fed 处于加息末期且通胀已回落，政策反应空间足；若危机发生在加息周期早期或通胀失控阶段，政策兜底速度会显著慢。

### E. 监管 / 禁令 / 交易通道冲击

区分：

- 禁令：短期不确定性和流动性冲击。
- 清晰监管框架：可能降低机构进入门槛。
- 交易所/托管限制：直接影响 BTC 资金通道。

观察：

- ETF 流向是否受影响。
- 价格是否在 SPX 平稳时独立下跌。
- 交易所、托管、稳定币是否有外部风险信号。

## 时间线模板

| 阶段 | 时间 | 读盘重点 |
|---|---|---|
| 即时反应 | T+0 到 T+3 | 流动性、价差、相关性可能失真 |
| 信息消化 | T+1 到 T+7 | 区分法币信任冲击 vs 实体经济冲击 |
| 政策反应 | T+7 到 T+30 | Fed、财政、监管回应决定中期方向 |
| 结构变化 | T+30 以后 | 储备资产叙事、供应链、资本管制是否持续 |

短期价格反应不等于中期方向。

---

## 异常组合速查

| 面板/外部组合 | 可能解释 |
|---|---|
| BTC 单日大跌 + SPX 平稳 | 加密内部或监管事件优先排查 |
| BTC/SPX 同跌 + DXY/VIX 上升 | 系统性去风险 |
| 黄金涨 + BTC 涨 + SPX 弱 | 非主权资产叙事增强 |
| 黄金涨 + BTC 跌 | BTC 未被当作避险资产，需谨慎 |
| ETF 大流出 + 监管新闻 | 机构通道受冲击 |

## 输出要求

引用本文时必须说明：

1. 事件属于哪一类。
2. 主要传导链是什么。
3. 哪些 guanfu 指标能验证，哪些需要外部数据。
4. 该地缘解释的失效条件。

若没有新闻上下文，不要凭指标臆造地缘事件。
