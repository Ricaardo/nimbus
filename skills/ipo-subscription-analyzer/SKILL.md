# IPO 打新分析师

## 核心功能

- **日历扫描** — 港股/美股/A 股三市场并行，获取当前可申购标的
- **9+1 维评分** — 基本面(20%) / 估值(12%) / 基石保荐人(18%) / 板块热度(15%) / 筹码稀缺性(5%) / 认购情绪(7%) / Red Flags(8%) / 市场时机(10%) / AH溢价(2%) / 治理(3%)
- **资金分配** — 根据资金规模自动适配甲乙组策略，含 10x 融资成本精算
- **回报测算** — 乐观/中性/悲观三情景，基于历史数据校准的中签率模型

## 数据获取

**实时 IPO 数据**（当前招股标的）优先用 futuapi `get_ipo_list()`，不可用时 fallback 到 WebSearch。
**历史对标数据**（暗盘、首日涨幅、基石等）全部来自 `ipo-historical-data.md` + 本次 WebSearch 更新。

详细的数据源策略、API 代码示例和端点表见 [`references/futu-api-guide.md`](references/futu-api-guide.md)。

## Workflow

### Step 0: 更新历史数据

每次运行先搜索最新上市标的，追加到 `ipo-historical-data.md`。搜索关键词按当前年份动态生成：`"{当前年份} 港股IPO 暗盘 首日涨幅 超购 一手盈利 国配"`。

提取 13 个字段（公司/代码/上市日/行业/入场费/超购/国配/基石占比/暗盘/首日/一手盈利/保荐人/基石投资者），追加到对应年度表格。缺失字段标「—」，后续补全。

如果趋势重大变化（连续 2 月平均偏离现有三情景 ±15%），同步更新下方速查基准表。

### Step 1: IPO 日历扫描

> ⚠️ **MANDATORY — 不可跳过**：Step 1 第一步必须调用 futuapi `get_ipo_list(Market.HK)`（和/或 `Market.US`），获取当前实际招股列表。WebSearch 存在 1-3 天延迟，会遗漏刚启动招股的标的。futuapi 返回的 `is_subscribe_status=True` 才是权威的"正在招股"判断。**仅在 futuapi 连接失败时才 fallback 到 WebSearch。**

```python
from futu import *
quote_ctx = OpenQuoteContext(host='127.0.0.1', port=11111)
ret, data = quote_ctx.get_ipo_list(Market.HK)  # 或 Market.US
# is_subscribe_status=True → 正在招股
# list_time → 上市日期
# entrance_price → 入场费
# ipo_price_min/max → 招股价区间
# lot_size → 每手股数
quote_ctx.close()
```

根据用户指令判断扫描范围：

| 用户指令示例 | 扫描范围 |
|-------------|---------|
| "港股打新"、"港股IPO"、"今日港股IPO分析" | 仅港股 |
| "美股IPO"、"US IPO"、"美股打新" | 仅美股 |
| "A股新股"、"A股打新"、"今日A股申购" | 仅A股 |
| "今日IPO分析"、"打新分析"、"IPO分析" | 全市场 |

每只标的提取：公司名称（中英文）、股票代码、招股价区间、每手股数、入场费、申购起止日期、定价日、暗盘日（港股）、上市日、保荐人/承销商、募资规模、当前状态。

### Step 2: 标的基本面分析

- 收入规模与增长率（近 3 年，>30% = 强劲，<10% = 疲弱）
- 收入质量（经常性 vs 一次性：授权首付款/资产出售是否占收入 >30%）⚠️ 常见粉饰套路
- 净利润/亏损与趋势
- 毛利率（>60% = 优秀，<30% = 偏弱）
- 市盈率/市销率 vs 已上市可比公司
- 业务模式与竞争壁垒（TAM、市场地位、护城河）
- 客户集中度风险（前 5 大客户收入占比）
- 现金流状况与烧钱速率（现金跑道 >24 月 = 健康）

### Step 3: Red Flags 检测

| 级别 | Red Flag | 说明 |
|------|----------|------|
| 严重 | 持续经营疑虑 | Going concern 警告 |
| 严重 | 内控重大缺陷 | Material weakness |
| 严重 | 监管调查/诉讼 | 正在进行中的重大法律风险 |
| 高 | 关联交易占比高 | >10% 收入来自关联方 |
| 高 | 内部人大量抛售 | 创始人/管理层借 IPO 套现 >30% |
| 高 | 国配<5x + 公开>1000x | 公开虚高国配不跟 → 破发率极高（优乐赛-44%、翰思艾泰-46%） |
| 高 | 募资用途模糊 | "一般公司用途"占比过高 |
| 中 | 双重股权结构 | 创始人超级投票权 |
| 中 | 关键客户流失 | 近期失去重要客户 |
| 中 | 对赌/赎回压力 | IPO必须在特定期限内完成否则触发赎回 |

严重 Red Flag 标的直接降至 <40 分。出现 2 个以上高级别 Flag 也建议回避。

> 国配<5x + 公开>1000x 的组合是历史数据中最精确的破发先行指标，不可因其不在传统 Red Flag 框架中而忽略。国配倍数体现了机构真实定价意愿，公开超购反映散户情绪——两者背离 = 危险。

募资用途：研发/产能扩张 = 正面，偿还债务 = 中性偏负，"一般公司用途" = 负面。

### Step 4: 板块与长期价值

- 所属板块近期市场热度（涨跌幅，资金流向）
- 行业生命周期阶段（萌芽/成长/成熟/衰退）
- 可比上市公司估值区间（P/E、P/S、EV/EBITDA）
- 政策环境（监管利好/利空）
- 板块热度评分：高热 / 中等 / 冷门

### Step 5: 基石投资者与保荐人评估

**基石投资者评分 (0-10)：**

| 类型 | 分值 | 示例 |
|------|------|------|
| 顶级主权基金/长线基金 | 8-10 | GIC、淡马锡、Capital Group、Fidelity、高瓴 |
| 知名产业资本/战略投资者 | 6-8 | 行业龙头、上下游巨头、大摩MSIP |
| 中型机构/家族办公室 | 4-6 | 区域基金、知名家办 |
| 关联方/短期对冲基金 | 1-3 | 控股股东关联公司、短线基金 |
| 无基石投资者 | 0 | — |

加分项：锁定期 >12 个月 (+1)、基石认购占比 >50% (+1)

**保荐人分级评分：**

| 级别 | 代表机构 | 加分 | 说明 |
|------|---------|------|------|
| **S 级** | **中信证券** | **+5** | 暗盘破发率 ~15%，历史胜率远超同行 |
| A 级 | 摩根大通、高盛、中金、华泰 | +3 | 国际/国内头部投行 |
| B 级 | 国泰君安、建银国际、海通、招银 | +1 | 中上水平 |
| C 级 | 其他中小券商 | 0 | 无额外加分 |

中信证券单独列为 S 级是因为历史数据显示其保荐项目暗盘及首日胜率显著优于同行——2026 年 7 个项目仅 1 个破发。

**基石质量比占比更重要。** 贝莱德/GIC/高瓴/淡马锡等顶级长线基金站台的价值远大于关联方凑数。历史数据：基石 >30% 的标的暗盘胜率约 85%。

### Step 6: 打新机制分析

#### 港股

**甲乙组分组**：≤500 万港元 = 甲组（抽签机制），>500 万 = 乙组（保证至少 1 手）

**入场费** = 每手股数 × 招股价上限 × 1.01（含 1% 手续费）

**融资**：富途和老虎均支持最高 10x 杠杆。富途多数标的免息，实际融资成本为 0。详见资金分配系统。

#### 美股

- 承销商阵容（Goldman Sachs/Morgan Stanley/JPMorgan = 顶级）
- 定价区间 vs 预期最终定价方向
- 散户可参与渠道（富途/老虎/盈透）
- 锁定期到期风险（通常 90-180 天）

#### A 股

- 网上申购上限、发行市盈率 vs 行业平均（>23 倍需关注）
- 中签缴款日、上市日
- 预估开板涨幅

### Step 6.5: 筹码稀缺性评估

小盘+货量稀缺是港股打新核心 alpha 来源，须单独评分（占 5%）：

| 信号 | 加分 | 说明 |
|------|------|------|
| H股占比 < 总股本 15% | +1 | 流通盘极小，供需紧张 |
| 募资总额 < 15亿港元 | +2 | 绝对值小盘，易被资金撬动 |
| 公开发售初始 ≤10% | +1 | 机制B/18C/18A，散货紧张 |
| 有绿鞋（超额配股权） | +1 | 稳价机制，暗盘/首日抛压缓冲 |

5 分 = 全部满足（如曦智：H股15%、募资25亿→4分；天星：H股15%、募资8亿→5分）

### Step 7: 资金冲突与时间表

标注申购期重叠（资金无法复用），计算资金占用天数，输出建议申购优先级。

### Step 8: 综合评分与建议

#### 评分卡（0-100 分）

| 维度 | 权重 | 评分依据 |
|------|------|---------|
| 基本面质量 | 20% | 盈利能力、增长性、现金流 |
| 基石/保荐人质量 | **18%** | 基石评分(0-10) + 保荐人分级加分（S级+5, A级+3, B级+1） |
| **板块热度** | **15%** | 板块涨跌幅、资金流向、政策面。医药/AI板块自动上浮，A+H自动下浮 |
| 估值合理性 | 12% | vs 可比公司折价/溢价 |
| 市场时机 | 10% | 大盘环境、近期 IPO 平均首日表现 |
| Red Flags 扣分 | 8% | 严重 Red Flag → <40；国配<5x+公开>1000x 视为高级别 |
| 认购热度/GMP 情绪 | 7% | 超额认购倍数、暗盘溢价、公开认购占比 |
| **筹码稀缺性** | **5%** | H股占比<15%/+1, 募资<15亿/+2, 公开初始≤10%/+1, 有绿鞋/+1 |
| 募资用途与治理 | 3% | 用途合理性 + 治理结构（对赌协议扣分） |
| A/H 溢价机会 | 2% | AH 定价差异套利空间 |

**建议等级**：≥75 = 强烈推荐 · 50-74 = 可适量申购 · <50 = 不建议

> **中信保荐地板规则**：中信证券保荐的标的，除非出现严重 Red Flag，否则总分不低于 50 分。历史依据：2026 年中信 7 个项目仅 1 个破发，最低首日表现极视角仍 +150%。

### Step 9: 暗盘捞货分析（如有今日暗盘标的）

已上市/已截止招股的标的若当日处于暗盘交易时段，评估是否值得在暗盘买入。

**五大看点：**

| 看点 | 具体看什么 | 决策意义 |
|------|-----------|---------|
| **价** | 暗盘价 vs 发行价 vs 同板块可比首日涨幅中位数 | 涨幅 < 中位数 → 可能低估；涨幅 > 上沿 → 已透支 |
| **量** | 换手率，尤其开盘前 30 分钟成交量 | 高换手 → 机构在吃货/出货；低换手 → 散户互搏，方向不可靠 |
| **时序** | 开盘冲高回落 vs 低开慢爬 vs 全天横盘 | 冲高回落 → 抛压大，暗盘卖优先；低开慢爬 → 资金在吸筹 |
| **对比** | 富途 vs 辉立 vs 耀才三个暗盘价差 | 价差 >5% → 流动性差/分歧大；价差 <2% → 共识强 |
| **情绪锚** | 当日大盘指数 + 同板块涨跌幅 | 大盘拖累暗盘弱 → 可能错杀；大盘强暗盘弱 → 真弱 |

**捞货决策框架：**

| 条件 | 操作 | 说明 |
|------|------|------|
| 暗盘价 < 发行价 × (1 + 可比首日中位涨幅) 且 无严重Red Flag | ✅ 可捞 | 定价未充分反映预期，首日修复概率大 |
| 暗盘破发（跌>0%）但 基石>30% + 国配>5x | ⚠️ 谨慎捞 | 机构未跑，可能是情绪/大盘错杀，轻仓+止损-5% |
| 暗盘破发 且 国配<5x 或无基石 | ⛔ 不碰 | 机构用脚投票，暗盘跌是定价回归而非错杀 |
| 暗盘涨幅 > 可比首日涨幅上沿 | ⛔ 不追 | 涨幅已透支，首日大概率高开低走 |
| 暗盘涨幅 > +100% 且 换手率 >30% | ⛔ 不追 | 筹码高度分散，首日抛压巨大 |

**捞货目标价位**：
```
目标卖出价 = 发行价 × (1 + 同板块可比首日均值涨幅)
止损价 = max(暗盘买入价 × 0.95, 发行价 × 0.95)
```

**核心铁律**：

> 1. **暗盘涨了大概率首日不如暗盘**——58.7% 的标的暗盘收盘价 > 首日收盘价。暗盘大涨应卖出而非追涨。
> 2. **暗盘是捡漏工具不是追涨工具**——正确场景是"暗盘小跌/持平但基本面+基石没毛病的标的被情绪错杀"。
> 3. **持有窗口仅到次日**——暗盘买入 → 首日开盘/收盘卖出，不扛过夜。暗盘捞货本质是 T+1 超短线博弈。
> 4. 暗盘捞货需结合 `technical-analysis` 实时 K 线判断买点，结合 `market-pulse` 判断当日大盘情绪。

## 资金分配系统

### 资金规模档位

| 档位 | 自有资金 | 融资倍数 | 策略 |
|------|---------|---------|------|
| 小额 | <10万 | 1-3x | 甲组 1 手现金 |
| 中额 | 10-50万 | 3-5x | 甲组为主 |
| 大额 | 50-100万 | 5-10x | 甲乙组结合 |
| 超大额 | >100万 | 10x+ | 乙组为主，多账户分散 |

### 券商融资参数

| 券商 | 利率 | 最高倍数 | 活动特惠 |
|------|------|---------|---------|
| 富途 | 3.8% p.a. | **10x** | 0%利息（多数标的免息） |
| 老虎 | 4.5% p.a. | 10x | 0%利息（部分标的） |
| 盈透 | 6.3% p.a. | 4x | 无 |
| 微牛 | 3.99% p.a. | 5x | 无 |

融资成本 = 融资额 × (年利率/365) × 资金占用天数。资金占用天数 = 申购截止日 - 申购日 + 3~5 天退款周期。

### 分配逻辑

优先顺序：顶级基石标的（乙组冲击，50%资金）→ 基本面稳健标的（甲组 5x 融资，30%）→ 赛道热门标的（甲组少量，10%）→ 高风险标的（0-5% 或观望）

分配矩阵和风险控制参数详见下方。

| 标的类型 | 资金占比 | 融资 | 策略 |
|---------|---------|------|------|
| 顶级基石 + 热门赛道 | 50% | 10x | 乙组冲击 |
| 基本面稳健 + 已盈利 | 30% | 5x | 甲组高位 |
| 赛道热门 + 盈利不稳 | 10% | 3x | 甲组少量 |
| 数据不足 + 风险较高 | 0-5% | — | 观望 |

风险阈值：单一标的融资 >60% 总融资 → 分散；破发 >50% → 止损；资金占用 >15 天 → 重评机会成本。

## 回报测算模型

### 每手收益（1 手为单位，不拆分）

```
每手收益 = (暗盘/首日收盘价 - 招股价) × 每手股数 - 手续费(约1%)
```

### 三情景预估

> ⚠️ **必须对标历史**：选取情景前，先在 `ipo-historical-data.md` 中找到 **3 个以上最可比标的**（同板块+同保荐人级别+同基石质量），用它们的实际首日涨幅校准三情景，而非机械套用下表。尤其是当前市场环境与历史基准出现 >15% 偏离时（如 2026 年 4 月均值 +138% vs 2025 年均值 +38%），应以最近 1-2 月数据为主导。

| 情景 | 适用条件 | 预期涨幅 |
|------|---------|---------|
| **乐观** | 基石顶级（贝莱德/GIC/高瓴级别）+ 板块火热 + 超购 >1000x | +40% ~ +80% |
| **中性** | 基石较好 + 市场平稳 + 基本面尚可 | +20% ~ +40% |
| **悲观** | 普通标的 + 市场调整 或 基本面差 + 基石弱 | -10% ~ +10% |

> 2026 年 4-5 月参考锚点：近 10 只新股首日均值 +59%，4 月均值 +138%。AI/硬科技板块首日常见 +40~80%（曦智+384%、商米+241%、海致+242%）。生物医药 +69%（2025 均值）。A+H 二次上市 +10~20%。在当前市场热度下，顶级基石+中信保荐的标的应优先匹配**乐观情景**。

### 中签率估算

甲组 1 手中签率 ≈ 50% / (超购倍数 × 90%)

多手中签概率 = 1 - (1 - 单手中签率)^认购手数

甲组是抽签机制——中签结果是 0 手或 1 手，不能拆分。预期收益 = 中签概率 × 1 手收益。

乙组保证至少获配 1 手，但热门标的（>1000x）可能只获配 1 手。

### 回报计算

```
预期收益 = 中签概率 × 1手收益
乐观预期 = 中签概率 × (乐观涨幅 × 入场费)
中性预期 = 中签概率 × (中性涨幅 × 入场费)
悲观预期 = 中签概率 × (悲观涨幅 × 入场费)
```

## 历史数据参考

> **详见**: [`ipo-historical-data.md`](ipo-historical-data.md) — 2025-2026 每只标的的 13 字段明细（暗盘、一手盈利、国配、基石占比、入场费等），每次 Step 0 自动更新。

### 速查基准

| 板块/信号 | 基准值 | 含义 |
|----------|--------|------|
| 生物医药 | 首日平均 **+69%** | 打新最强赛道 |
| AI/硬科技 | 首日 +40~80% | 分化大，国配是关键 |
| 消费/零售 | 首日 +20~30% | 稳定但爆发力弱 |
| A+H 二次上市 | 首日 +10~20% | 折价小，涨幅有限 |
| 暗盘 >1000x 超购 | 胜率 **95.2%** | 暗盘几乎必赚 |
| 国配 <5x + 公开 >1000x | ⚠️ 破发高危 | 优乐赛、翰思艾泰都是这个模式 |

### 三情景校准

| 情景 | 涨幅 | 历史对照 |
|------|------|---------|
| 乐观 | +40~80% | 2026.4 月均值 +138%，AI/硬科技首日 +40~80%，医药 ~69%。顶级基石+中信保荐进入此档 |
| 中性 | +20~40% | 2026 Q1 均值 +32%，2025 全年均值 +38%。基石较好、基本面稳健、非顶级赛道标的 |
| 悲观 | -10~+10% | 2026 Q1 破发率 12.5%。国配薄弱+公开虚高、A+H折价有限、基本面差的标的 |

> **校准规则**：每次 Step 0 更新历史数据后，若近 10 只新股首日均值偏离当前三情景中位 >15%，自动调整各情景预期涨幅。当前锚点：近 10 只首日均值 +59.22%，处于乐观区间上沿。

## 输出格式

> **详见**: [`references/output-format.md`](references/output-format.md) — 标准报告模板、字数控制、中签率计算格式、Dashboard JSON schema。

### 核心原则

| 规则 | 说明 |
|------|------|
| **默认产出 MD 文件** | 每次分析写入 `reports/ipo_subscription_YYYYMMDD.md` |
| **结论先行** | 每只标的第一行给出评分+建议等级，细节在后 |
| **只保留可操作标的** | 已过暗盘/已上市的标的**不出现在报告中**（仅 futuapi 标记 `is_subscribe_status=False` 时跳过） |
| **对比理由后置** | 可比公司对标、评分推导过程放在报告末尾「📎 评分推导」折叠区 |
| **表格优先** | 招股日历、评分、资金分配、回报测算全部表格化 |
| **字数控制** | 每只正在招股标的 ≤150 字（结论先行节省大量篇幅），总报告 ≤1500 字 |
| **甲组中签** | 写「中签概率 XX%，中签则获 1 手」，不写分数手数 |

### 报告结构

```
# 港股 IPO 打新分析 YYYY-MM-DD

## 🎯 结论（≤100字）
[一句话推荐排序 + 核心原因]

## 📊 招股日历
[表格：仅 is_subscribe_status=True 的标的]

## 🔥 标的速评（每只 ≤150字）
[评分 → 亮点 → 风险 → 中签预估]

## 💰 资金分配
[表格]

## ⚡ 操作清单
[时间线表格]

## 🕵️ 暗盘捞货（如有今日暗盘标的）
[暗盘价 vs 发行价折溢价 + 是否值得捞 + 目标价位]

## 📎 评分推导
[可比对标 + 各维度得分明细]
```

中文输出，保留英文 ticker 和专有名词。

## 🔗 联动技能

| 场景 | 使用技能 | 关系 |
|------|---------|------|
| 分析个股基本面/估值 | `us-stock-analysis` | 标的深度分析 |
