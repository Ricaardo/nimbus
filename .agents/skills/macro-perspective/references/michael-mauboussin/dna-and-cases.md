# Michael Mauboussin — 表达 DNA + 案例库

## 🎙 表达 DNA

1. **学术风格**：每个论点引用学术文献（Kahneman, Tversky, Tetlock, Galton）
2. **跨学科**：心理学 + 概率论 + 演化生物学 + 体育统计
3. **基础率优先**：永远先问"历史 base rate 是什么"
4. **谨慎警句**：不下绝对结论，常用 "tend to" / "on average"
5. **教学优先**：他的写作目标是让读者更聪明，而非展示自己

**典型句式**：
- "What's the base rate?"
- "More than you know."
- "When luck plays a role, reversion to the mean is inevitable."
- "Skill is judged by process, not outcome."

## 🧠 核心框架 — 4 个

### 1. 运气-技能连续谱（Skill-Luck Continuum）

```
纯运气                                      纯技能
彩票 — 21点 — 扑克 — 投资 — 国际象棋 — 围棋
       (60% 运气)         (5% 运气)
```

**判别运气 vs 技能的 3 测试**：
- **Reversion to mean**：极端表现回归速度（运气越多回归越快）
- **可故意输掉？**：能就是技能多（围棋能，彩票不能）
- **样本一致性**：同一人多年持续优秀 = 技能；偶发 = 运气

### 2. 基础率思维（Base Rate Thinking）

**3 步必问**：
1. **outside view**：这类 case 历史平均结果是什么？
2. **inside view**：我手上的这个 case 有多特殊？
3. inside view 能否压过 outside view？（通常不能）

**经典基础率**：
- IPO 第一年：跑输大盘（base rate 70%+）
- 收购溢价：买方股价 -3% 至 -5% 平均
- 财报后漂移（PEAD）：beat 后 60 天超额 +1.5%
- 5 年 CEO 任期 ROIC 改善：< 30% 公司能做到

### 3. 期望投资（Expectations Investing）

**反向工程市场预期**：
- 不要预测股价
- 反推：当前价格隐含的"市场预期是什么"
- 你的预期 vs 市场预期，差异 = alpha source

**Mauboussin 的标准 5 步**：
1. 估当前股价隐含的销售增长率
2. 估隐含的利润率
3. 估隐含的资本效率
4. 比较你的判断 vs 隐含
5. 差异 > 30% → 信号

### 4. 多样性预测（Wisdom of Crowds 的边界）

**3 条件**让群体判断有效：
1. 多样性（不同信息 / 不同视角）
2. 独立性（不互相影响）
3. 聚合机制（市场价格、投票）

**3 条件**让群体失败（变成 herd）：
1. 信息层级（全部跟领先者）
2. 社会模仿（怕错失 / FOMO）
3. 反馈循环（价格上 → 更多人买 → 价格更上）

→ Mauboussin 认为这是泡沫的标准触发条件

## 📕 局限

### 框架不直接产生交易信号
- Mauboussin 的工具是"思维改进"，不是"买什么"
- 需要和具体投资策略结合才能落地

### 基础率失效的两类情况
- **极端事件**：黑天鹅，base rate 不适用
- **结构性变化**：base rate 本身在变（如 2020 后 IPO 锁定期变化）

## ⚠ 常见误用

- **"Mauboussin 推荐了 X"** — 几乎从不。他不做个股推荐
- **把基础率当死规则** — 错。他强调"知道何时该 override"
- **把"运气 vs 技能"用于贬低别人** — 错用。他自己说要先用在自身
- **期望投资套个股不查市场结构** — 错。需要先验证市场是否真在 price-in 那些预期
