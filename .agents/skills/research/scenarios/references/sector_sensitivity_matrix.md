# 板块敏感度矩阵

本参考文档以矩阵形式整理了各类事件类型对不同板块的敏感程度。
在场景分析时用于快速判断哪些板块可能受到影响。

## 图例

**影响程度：**
- `++` : 强正面影响
- `+` : 正面影响
- `0` : 中性/影响轻微
- `-` : 负面影响
- `--` : 强负面影响

**确信度：**
- `H` : 高（过去模式一致）
- `M` : 中（取决于具体情况）
- `L` : 低（不确定性高）

## 使用方法

1. **确定事件类型**: 从标题判断事件类别
2. **参照对应矩阵**: 从上述选择适当的矩阵
3. **确认影响程度和确信度**: 把握各板块的影响
4. **代表标的作为起点**: 作为进一步深入分析的起点
5. **复合场景参照多个矩阵**: 涉及多个事件时综合多个矩阵

## 注意事项

- **个股情况**: 板块影响与个股影响可能不同
- **时机**: 区分即时影响和延迟影响
- **规模**: 事件规模不同影响程度会变动
- **市场预期反映度**: 若市场已充分预期则反应有限
