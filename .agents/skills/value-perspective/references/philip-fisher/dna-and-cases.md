# Philip Fisher — 表达 DNA + 案例库

## 🎙 表达 DNA（成长投资之父，5 个标记）

1. **Scuttlebutt（闲聊调研）**：真相在公司外——问客户/供应商/竞争对手/前员工，拼出管理层不会告诉你的全貌。
2. **质量 > 便宜**：宁买卓越成长以合理价，不买平庸以便宜价。极少卖出。
3. **定性优先**：管理层诚信、研发文化、销售组织这些"软"因素，比当期 PE 重要。
4. **极度集中 + 极长持有**：一生只持少数票，持有以"十年"计（Motorola 持了 ~40 年到死）。
5. **逆共识买成长**：在卓越成长股**暂时失宠**时下手。

**典型句式**：
- "The best time to sell a stock is almost never."
- "I don't want a lot of good investments; I want a few outstanding ones."
- "Conservative investors sleep well."

## 🧠 核心框架

**15 要点(15 Points to Look For)**——一家值得长持的成长公司应满足（精选关键项）：
1. 产品/服务有**足够长的成长跑道**（不是一次性爆发）
2. 管理层有**持续开发新产品**的决心（研发文化）
3. 研发**相对规模的有效性**（投入产出，不是烧钱）
4. **高于行业的利润率** + 有**维持/提升利润率**的措施
5. 卓越的**销售组织**（常被忽视，却是成长落地的关键）
6. 优秀的**劳资/高管关系** + 管理层**深度**（不靠一个人）
7. 管理层**对股东诚实**（坏消息也讲）
8. 长期看是否需要**频繁增发稀释**（不需要 = 加分）

**Scuttlebutt 方法**：系统性访谈产业链上下游与离职员工，验证管理层说辞、发现非共识。

**三个（也是唯一三个）卖出理由**：
1. 当初**判断错了**（基本面认知有误）；
2. 公司**基本面恶化**（不再符合 15 要点）；
3. 发现**显著更好的机会**（机会成本）。
—— 注意：**"涨太多了/估值高了"不在卖出理由里**。

## 📚 案例

### Motorola（1955 买入，持有至 2004 去世）
- Fisher 的标志性长持，近半世纪，复利惊人。
- 逻辑：研发文化 + 管理层深度 + 不断进入新成长曲线（半导体→通信）。

### Texas Instruments / Dow Chemical
- 同样的"研发驱动成长 + 长持"逻辑下的重仓。

## 📕 局限 / 错误

- **过度集中风险**：极少数票，单一管理层/技术路线判断错则伤筋动骨。
- **"几乎不卖"被误用**：他指的是**卓越公司**；普通公司套用"持有forever"=灾难。
- 对**估值不敏感**：买入价偏贵时，再好的成长也可能多年不赚钱（需与 Graham/Klarman 的价格纪律互补）。

## ⚠ 常见误用

- **Scuttlebutt ≠ 刷 Reddit/雪球**：真正的 scuttlebutt 是结构化的产业链访谈，不是看散户情绪。
- **"成长就该贵"**：Fisher 要"合理价的卓越成长"，不是"任何价的故事股"。
- **无视卖出纪律的前两条**：很多人只记"几乎不卖"，忘了"判断错/基本面坏要走"。
