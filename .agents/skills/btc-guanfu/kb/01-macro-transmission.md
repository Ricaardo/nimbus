# 宏观传导 (Macro Transmission)

> 用于解释 `macro` 域为什么影响 BTC。实时数值以 `guanfu --json` 为准；本文只给传导链、判别逻辑和失效条件。

---

## 适用时机

当以下任一条件出现时加载本文：

- `macro.real_yield_10y_pct` 接近或突破关键阈值。
- `macro.dxy_60d_trend_pct` 绝对值扩大。
- `macro.m2_yoy` 转向或跨越 0%、3%、5%。
- `macro.spx_correlation_30d` 升到高相关或跌到弱相关。
- `macro.hy_spread_bps` / `macro.yield_curve_10y2y_bps` 显示信用或衰退压力。

---

## 利率传导链

```
Fed 政策利率
  → 短端利率：现金/货币基金机会成本
  → 长端利率：企业融资、估值分母、风险偏好
  → 实际利率：持有无现金流资产的真实机会成本
  → 利率曲线：衰退预期和信用条件
  → BTC 风险溢价和资金流
```

### `macro.real_yield_10y_pct`

实际利率是 BTC 最重要的宏观变量之一，因为 BTC 不产生现金流。实际利率越高，持有现金/债券的相对吸引力越强。

| 区间 | 解释 | BTC 读盘含义 |
|---|---|---|
| `<0%` | 现金真实收益为负 | 强顺风，但仍要看杠杆拥挤 |
| `0-1%` | 宽松到温和 | 偏顺风 |
| `1-2%` | 有机会成本 | 中性偏逆风，可被 ETF/内部催化剂抵消 |
| `2-2.5%` | 高机会成本 | 逆风增强，估值信号需降权 |
| `>2.5%` | 极高机会成本 | 宏观逆风优先级高于普通低估信号 |

### 时滞

利率政策到 BTC 通常不是即时传导。常见路径是：

```
政策转向 → 债券/美元先动 → 股票和信用再动 → 加密资金流最后确认
```

一般观察窗口是数周到数月。单次 FOMC 或单日利率变动只作触发，不直接当作趋势。

## 通胀传导链

通胀对 BTC 不是单向利好或利空。要先判断通胀来源：

| 类型 | 来源 | 资产反应 | BTC 含义 |
|---|---|---|---|
| 需求拉动 | 经济强、工资强、刺激政策 | 风险资产和商品同涨 | 偏顺风 |
| 成本推动 | 能源、供应链、关税 | 股票利润承压，央行更难放松 | 中期偏逆风 |
| 滞胀 | 增长弱 + 通胀高 | 股票/债券可能双杀 | 不确定性最高 |

读盘时不要只看 CPI 或油价。更关键的是 Fed 是否会被迫鹰派，以及实际利率是否因此上升。

## 美元传导链

`macro.dxy_60d_trend_pct` 表示贸易加权美元 60 日趋势。美元走强通常压制 USD 计价的 BTC，但原因比绝对值更重要。

| 美元走强来源 | 识别线索 | BTC 含义 |
|---|---|---|
| 利差驱动 | 美国利率相对上升 | 偏逆风 |
| 避险驱动 | DXY、VIX、黄金同时上升 | 短期风险资产承压，中期看危机性质 |
| 贸易/关税驱动 | 关税、贸易摩擦、输入性通胀 | 可能形成美元和通胀双逆风 |
| 去美元化反身性 | DXY 弱、黄金强、法币信任下降 | BTC/黄金叙事增强 |

不要混用 DXY 和 FRED `DTWEXBGS` 的绝对点位阈值。guanfu 的主指标是 60 日变化率；若要讨论绝对美元指数，必须说明使用的是哪个序列。

## 货币供应与流动性

`macro.m2_yoy` 是慢变量，常滞后发布（月频 + 1-2 个月口径滞后）。它适合判断背景流动性，不适合单独做短期择时。

阈值是按 1990-2025 美国 M2 YoY 历史分布校准的（中位约 5%、COVID 峰值 27% 是异常值，不代表常规上限）：

| `macro.m2_yoy` | 含义 | BTC 读盘 |
|---|---|---|
| `< 0%` | 货币供应收缩（罕见，2009、2022-23 短窗口） | 逆风，低估信号需等待确认 |
| `0-3%` | 停滞或温和 | 流动性偏紧 |
| `3-6%` | 中位附近 | 中性偏顺风 |
| `6-10%` | 扩张 | 顺风 |
| `> 10%` | 极端扩张（仅 COVID 类危机刺激出现） | 强顺风，但通常伴随通胀反扑风险，需同时看过热 |

高频确认优先看 ETF 净流入、稳定币 30 日增速、funding/OI、美元趋势。

## 信用与曲线

### `macro.hy_spread_bps`

HY spread（ICE BofA HY OAS, FRED `BAMLH0A0HYM2`）是信用市场压力计。两类阈值分开看：

**绝对水平**（背景压力）：
- `< 350bp`：风险偏好较好。
- `350-500bp`：正常到偏紧。
- `500-800bp`：信用紧缩，风险资产需降置信。
- `> 800bp`：危机背景区。
- 历史参考峰值：2008 ~2200bp，2020-03 ~1100bp，2022-09 ~600bp。

**变化速度**（事件触发）：
- 单日扩大 > 50bp 或 5 日扩大 > 100bp：信用市场再定价进行中。

进入危机模式的具体触发以 `kb/09-crisis-playbook.md` 为准（绝对水平 + 变化速度任一即可）。

### `macro.yield_curve_10y2y_bps`

收益率曲线（FRED `T10Y2Y`）用于识别增长预期，**不直接决定 BTC 方向**：

- `< 0bp`：倒挂，衰退风险上升。历史上倒挂到衰退确认通常滞后 12-24 个月，且不必然导致 BTC 立即下跌（2022-23 倒挂期间 BTC 仍走出独立行情）。
- `0-100bp`：正常化或过渡期。
- `> 100bp`：陡峭化，需判断是复苏驱动（短端下移）还是长端失控（长端上行 + 财政担忧）。

曲线必须结合 `real_yield_10y_pct`、`hy_spread_bps`、`dxy_60d_trend_pct` 一起读。

## SPX 相关性

`macro.spx_correlation_30d` 告诉你 BTC 当前更像宏观风险资产，还是在走自身叙事。

| 相关性 | 含义 | 应看什么 |
|---|---|---|
| `< 0` | 独立/反相关 | ETF、链上、监管或加密内部叙事 |
| `0-0.3` | 弱相关 | BTC 自身因素更重要 |
| `0.3-0.7` | 中等相关 | 宏观和加密内部共同作用 |
| `> 0.7` | 强风险资产联动 | SPX、美元、实际利率权重上升 |

相关性不是方向，只是"解释变量权重"。

## 组合判别

优先使用组合，不用单点：

- `real_yield > 2.5%` + `m2_yoy < 0`：宏观逆风主导。
- `real_yield < 1%` + `m2_yoy > 5%` + `dxy_60d < -1%`：宏观顺风主导。
- `real_yield 1-2%` + ETF 流入强：宏观逆风可被边际需求抵消。
- `DXY 上行` + `HY spread 扩大` + `SPX corr > 0.7`：系统性去风险，降低常规类比权重。

## 输出要求

引用本文时，结论应写成：

- “宏观顺风/逆风/中性，证据是 A/B/C。”
- “该宏观结论的失效条件是 X。”
- “该结论置信度受数据时效/source health 限制。”

不要写成：

- “看到某阈值就买/卖。”
- “宏观变量单独决定 BTC。”
- “历史上一定如何。”
